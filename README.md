# Dark Mode Toggle 🌙☀️

A simple and modern dark mode toggle feature built using HTML, CSS, and JavaScript. This project allows users to switch between light mode and dark mode with a smooth and user-friendly interface.

## 🚀 Features
- Toggle between light mode and dark mode
- Smooth theme transition
- Responsive and minimal UI
- Theme preference saved using localStorage
- Clean and reusable code

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ⚙️ How It Works
The toggle button switches the website theme between light and dark modes. The selected theme is stored in the browser's localStorage so that the user's preference remains saved even after refreshing the page.

## 📦 Installation
1. Clone the repository
git clone https://github.com/sujaldhiman01/Dark-Mod

2. Open the project folder  
3. Run `index.html` in your browser

## 🎯 Purpose
This project demonstrates how to implement a theme switcher using JavaScript and store user preferences locally for a better user experience.
